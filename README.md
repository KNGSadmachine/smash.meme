# *smash.meme*
*smash.* の表示はなぜ迫力があるのか？

それを検証するiOSアプリのα版

## 【対応機種】

iPhone11, 11ProMax, 12, 12ProMax

上記機種以外は現在非対応．

おすすめはiPhone11です．


## 【アップデート履歴】

**[2021/02/15]**

UIの微修正をしました．

**[2021/02/13]**

開発者の画像上にsmash.のUIを表示できるようになりました．



## 【ベータ版以降での実装予定機能】

* メディアアイコンを選択することでカメラアプリの起動，撮影した写真の上にUIを表示

* iPhone内の画像や動画を選択して表示

* UIアイコンの表示，非表示の切り替え


    *and more...*
